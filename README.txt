CAMPO MINADO REVERSO:

A intenção do jogo é achar todas as bombas e conseguir a maior pontuação que puder (a máxima é 220). Você começa com 100 pontos e são 6 bombas a serem encontradas nas 81 casas elencadas de 0 a 80.

Para jogar: 

1) Clique na seta e depois vá para o método "mover_seta".
2) Diga qual casa deseja visitar para verificar se há bomba. 

Para ver pontuação: 

1) Clique na pontuação ao lado do campo.
2) Clique no método "exibir_pontuacao"

Para reiniciar: 

1) Clique no emoji em cima do campo minado.
2) Clique no método "reiniciar_jogo"

Para verificar quantas bombas restam: 

1) Clique na bomba em baixo do campo minado.
2) Clique no método "mostrar_bombas_restantes"
